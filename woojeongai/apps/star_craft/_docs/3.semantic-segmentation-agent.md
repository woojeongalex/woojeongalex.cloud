## 1. 모델 선택 — PSPNet 대신 SegFormer (RTX 3050 최적)

원서의 PSPNet은 ResNet-50/101 백본으로 RTX 3050 8GB에서 파인튜닝 시 VRAM 한계.
아래 모델 중 하나를 선택한다 (기본값: SegFormer-B0).

| 모델 | VRAM 추론 | VRAM 파인튜닝(QLoRA) | ADE20K mIoU | 지원 |
|------|-----------|---------------------|-------------|------|
| **SegFormer-B0** | ~1.5GB | ~3GB (배치 8) | 37.4 | HuggingFace ✅ 기본값 |
| SegFormer-B2 | ~3GB | ~5GB (배치 4) | 45.6 | HuggingFace ✅ |
| PSPNet-ResNet50 | ~4GB | ⚠️ 배치 2만 가능 | 41.7 | mmsegmentation |
| DeepLabV3+ | ~3.5GB | ~5GB (배치 2) | 45.0 | torchvision ✅ |
| Mask2Former (추천 대안) | ~5GB | ~7GB (QLoRA, 배치 2) | 56.1 | HuggingFace ✅ |

> **QLoRA 파인튜닝 목적이면 SegFormer-B2 추천** — B0 대비 mIoU +8, VRAM은 배치 4로 처리 가능.
> **최고 정확도 원하면 Mask2Former** — 단, VRAM 7GB 근접으로 다른 프로세스와 충돌 주의.

## 2. 구현할 기능

SegFormer-B0(또는 B2)로 이미지의 픽셀별 클래스를 예측하고,

- 분할 마스크를 컬러 오버레이 이미지로 시각화한다.
- 분할 결과(클래스별 픽셀 비율)를 **Neo4j Graph DB** 에 노드로 저장한다.
- 시각화된 마스크 이미지를 **S3** 에 업로드한다.
- FastAPI `POST /api/star-craft/segmentation/segment` 로 이미지를 수신한다.
- (선택) `POST /api/star-craft/segmentation/train` 으로 커스텀 데이터셋 파인튜닝.

### Step 5 — Tool 함수 (CPU/GPU-bound)

**`app/tools/segformer_segment_tool.py`**
```
- 싱글턴 패턴:
    _processor = None
    _model = None
    _model_lock = threading.Lock()

- def _get_model_and_processor():
    global _processor, _model
    with _model_lock:
        if _model is None:
            _processor = SegformerImageProcessor.from_pretrained(
                nvidia/segformer-b0-finetuned-ade-512-512
            )
            _model = SegformerForSemanticSegmentation.from_pretrained(
                nvidia/segformer-b0-finetuned-ade-512-512
            )
            device = cuda if torch.cuda.is_available() else cpu
            _model.to(device)
            _model.eval()
    return _processor, _model

- def segment_image(image_bytes: bytes, threshold: float) -> SegmentationVO:
    processor, model = _get_model_and_processor()
    image = Image.open(io.BytesIO(image_bytes)).convert(RGB)
    inputs = processor(images=image, return_tensors=pt).to(model.device)

    with torch.no_grad():
        outputs = model(**inputs)

    # 업샘플링: 원본 해상도로 복원
    logits = outputs.logits   # (1, num_labels, H/4, W/4)
    upsampled = torch.nn.functional.interpolate(
        logits, size=image.size[::-1], mode=bilinear, align_corners=False
    )
    pred_mask = upsampled.argmax(dim=1)[0].cpu().numpy()  # (H, W)

    # 클래스별 픽셀 비율 계산
    total_pixels = pred_mask.size
    classes_found = []
    for class_id in np.unique(pred_mask):
        ratio = float((pred_mask == class_id).sum() / total_pixels)
        if ratio >= threshold:
            label = model.config.id2label.get(int(class_id), fclass_{class_id})
            classes_found.append(SegmentClass(label=label, class_id=int(class_id), pixel_ratio=ratio))

    classes_found.sort(key=lambda x: x.pixel_ratio, reverse=True)
    dominant = classes_found[0].label if classes_found else unknown

    return SegmentationVO(
        classes_found=classes_found,
        dominant_class=dominant,
        image_width=image.width,
        image_height=image.height,
        mask_array_shape=(pred_mask.shape[0], pred_mask.shape[1]),
    ), pred_mask   # pred_mask는 시각화 툴로 전달

- CPU/GPU-bound → def segment_image(...) — async 절대 금지
- 반환 타입: tuple[SegmentationVO, np.ndarray]
```

### Step 6 — Interactor

**`app/use_cases/segmentation_interactor.py`**
```
- SegmentationInteractor(SegmentationUseCase) 구현
- __init__: model: SegmentationModelPort, graph: SegmentationGraphPort, storage: SegmentationStoragePort

- async def segment(self, cmd: SegmentCommand) -> SegmentResponse:
    1. (vo, pred_mask) = await asyncio.to_thread(
           segment_image, cmd.image_bytes, cmd.confidence_threshold
       )
    2. vis_bytes = await asyncio.to_thread(
           colorize_mask, pred_mask, cmd.image_bytes
       )
    3. url = await self.storage.save_mask_image(cmd.filename, vis_bytes)
    4. classes_dict = [{label: c.label, pixel_ratio: c.pixel_ratio}
                       for c in vo.classes_found]
    5. node_id = await self.graph.save_segmentation(
           cmd.filename, vo.dominant_class, classes_dict, url
       )
    6. return SegmentResponse(
           node_id=node_id,
           filename=cmd.filename,
           dominant_class=vo.dominant_class,
           classes_found=classes_dict,
           mask_image_url=url,
       )

- def train(self, cmd: TrainCommand) -> TrainResponse:
    # HuggingFace Trainer 기반 파인튜닝 — GPU-bound이므로 def (동기)
    # 1. SegformerForSemanticSegmentation.from_pretrained(..., num_labels=cmd.num_labels)
    # 2. 커스텀 데이터셋 로드 (images/ + masks/ 구조)
    # 3. Trainer(model, training_args, train_dataset, eval_dataset, compute_metrics)
    # 4. trainer.train()
    # 5. mean_iou = trainer.evaluate()[eval_mean_iou]
    # 6. self.model.save(trainer.state.best_model_checkpoint)
    # 7. return TrainResponse(...)

- adapter/inbound 스키마 import 금지
```

## 6. 안티패턴 — 하지 말 것

```python
# ❌ 추론 함수에 async 붙이기 (GPU-bound)
async def segment_image(image_bytes: bytes): ...

# ❌ pred_mask(numpy)를 DTO에 직접 담기 — 직렬화 불가
@dataclass(frozen=True)
class SegmentResponse:
    mask_array: np.ndarray   # 절대 금지 — URL로 대체

# ❌ 시각화 툴이 SegmentationVO 내부 구현에 의존
def colorize_mask(vo: SegmentationVO):   # VO 전달 금지 — np.ndarray만 받을 것

# ❌ 3개 포트를 하나로 합치기 (ISP 위반)
class SegmentationPort(ABC):
    def save(...): ...
    async def save_segmentation(...): ...
    async def save_mask_image(...): ...

# ❌ 모델을 요청마다 새로 로드
```
