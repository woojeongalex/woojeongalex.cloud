# GAN 이미지 생성 에이전트 구현 지시서 (DCGAN / Self-Attention GAN)

> 대상: `woojeongai/apps/star_craft/`
> 패턴 정본: `woojeongai/CLAUDE.md` §3–5, `star_craft/dependencies/yolo_provider.py`, `star_craft/adapter/outbound/resource_adapters/yolo/yolo_model_adapter.py`
> 하드웨어: RTX 3050 8GB VRAM / RAM 16GB / i7-8700
> 완료 기준: 하네스 게이트(`ruff check`, `ruff format`, `mypy`) 전부 통과

---

## 0. 구현 전 필독

1. `woojeongai/CLAUDE.md` 전문을 읽고 4계층·ISP·DIP·Director 패턴을 숙지한다.
2. **패턴 정본** — 아래 파일을 반드시 먼저 읽어 기존 스타일을 맞춘다.
   - `star_craft/dependencies/yolo_provider.py`
   - `star_craft/adapter/outbound/resource_adapters/yolo/yolo_model_adapter.py`
   - `star_craft/app/use_cases/yolo_interactor.py`
3. 기존 파일을 **절대 수정하지 않는다**. 신규 파일만 추가한다.
4. 모든 파일 상단에 `from __future__ import annotations` 를 붙인다.
5. GAN 학습·생성은 GPU-bound → `def` (동기), 호출 측에서 `asyncio.to_thread` 위임.
6. GAN은 다른 에이전트와 달리 **추론(생성)과 학습이 동등하게 중요**하다.
   Generator 가중치가 없으면 생성 불가 — 반드시 학습 후 생성 엔드포인트 호출.

## 1. 모델 선택 — DCGAN과 SAGAN 중 선택

원서의 두 모델을 모두 지원하되, 기본값은 DCGAN (구현 단순, 8GB VRAM 여유).

| 모델 | VRAM 학습 | VRAM 생성 | 특징 | 권장 해상도 |
|------|-----------|-----------|------|-------------|
| **DCGAN** | ~2GB (배치 64) | ~0.5GB | 구현 단순, 학습 안정 | 64×64 기본값 |
| **SAGAN** | ~4GB (배치 32) | ~1GB | Self-Attention으로 전역 구조 포착 | 128×128 권장 |
| StyleGAN2 (대안) | ~7GB (배치 8) | ~2GB | 최고 품질, 학습 복잡 | 256×256 |

> **RTX 3050 8GB 제약**: StyleGAN2는 VRAM 7GB 근접 — 다른 프로세스 병행 불가.
> 단계적 접근: DCGAN으로 먼저 파이프라인 검증 → SAGAN으로 품질 향상.

## 2. GAN 핵심 구조 (구현 전 숙지)

```
Noise z (latent_dim=100)
    ↓
Generator (G) → 가짜 이미지
    ↓                   ↑
Discriminator (D) ← 진짜 이미지 (학습 데이터)
    ↓
진짜/가짜 판별 → Loss → G·D 교대 업데이트
```

- **Generator**: z → 이미지 (ConvTranspose2d 업샘플링)
- **Discriminator**: 이미지 → 진짜 확률 (Conv2d 다운샘플링)
- **학습 루프**: D 업데이트 → G 업데이트 (교대)
- **SAGAN 추가**: Self-Attention 레이어 (G·D 중간 레이어에 삽입)

---

## 3. 구현할 기능

DCGAN 또는 SAGAN으로,

- `POST /api/star-craft/gan/train` — 커스텀 이미지 데이터셋으로 G·D 학습
- `POST /api/star-craft/gan/generate` — 학습된 Generator로 이미지 N장 생성
- 생성된 이미지를 **S3** 에 업로드하고 URL 목록 반환
- 학습 진행 샘플(epoch별 고정 노이즈 생성 결과)을 **S3** 에 저장
- 생성 메타데이터를 **Neo4j Graph DB** 에 노드로 저장

### Step 5 — 모델 정의 (신규 파일)

**`app/tools/gan_models.py`**
```
DCGAN Generator:
class DCGANGenerator(nn.Module):
    def __init__(self, latent_dim: int, image_size: int, channels: int = 3):
        # latent_dim=100, image_size=64 기준
        # ConvTranspose2d 레이어 4개: 4→8→16→32→64
        # 각 레이어: ConvTranspose2d → BatchNorm2d → ReLU
        # 마지막: ConvTranspose2d → Tanh (출력 -1~1)
        ngf = 64   # feature map 기본 채널
        self.net = nn.Sequential(
            # z (latent_dim,1,1) → (ngf*8, 4, 4)
            nn.ConvTranspose2d(latent_dim, ngf*8, 4, 1, 0, bias=False),
            nn.BatchNorm2d(ngf*8), nn.ReLU(True),
            # (ngf*8,4,4) → (ngf*4,8,8)
            nn.ConvTranspose2d(ngf*8, ngf*4, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf*4), nn.ReLU(True),
            # (ngf*4,8,8) → (ngf*2,16,16)
            nn.ConvTranspose2d(ngf*4, ngf*2, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf*2), nn.ReLU(True),
            # (ngf*2,16,16) → (ngf,32,32)
            nn.ConvTranspose2d(ngf*2, ngf, 4, 2, 1, bias=False),
            nn.BatchNorm2d(ngf), nn.ReLU(True),
            # (ngf,32,32) → (channels,64,64)
            nn.ConvTranspose2d(ngf, channels, 4, 2, 1, bias=False),
            nn.Tanh(),
        )
    def forward(self, z): return self.net(z)

DCGAN Discriminator:
class DCGANDiscriminator(nn.Module):
    def __init__(self, image_size: int, channels: int = 3):
        ndf = 64
        self.net = nn.Sequential(
            # (channels,64,64) → (ndf,32,32)
            nn.Conv2d(channels, ndf, 4, 2, 1, bias=False), nn.LeakyReLU(0.2, True),
            # → (ndf*2,16,16)
            nn.Conv2d(ndf, ndf*2, 4, 2, 1, bias=False), nn.BatchNorm2d(ndf*2), nn.LeakyReLU(0.2, True),
            # → (ndf*4,8,8)
            nn.Conv2d(ndf*2, ndf*4, 4, 2, 1, bias=False), nn.BatchNorm2d(ndf*4), nn.LeakyReLU(0.2, True),
            # → (ndf*8,4,4)
            nn.Conv2d(ndf*4, ndf*8, 4, 2, 1, bias=False), nn.BatchNorm2d(ndf*8), nn.LeakyReLU(0.2, True),
            # → (1,1,1) 스칼라 확률
            nn.Conv2d(ndf*8, 1, 4, 1, 0, bias=False), nn.Sigmoid(),
        )
    def forward(self, x): return self.net(x).view(-1)

SAGAN Generator / Discriminator:
- DCGANGenerator/Discriminator 상속 후 중간 레이어에 SelfAttention 삽입
- Generator: 16×16 해상도 레이어 다음에 SelfAttention 추가
- Discriminator: 16×16 해상도 레이어 다음에 SelfAttention 추가

def build_generator(model_type: str, latent_dim: int, image_size: int) -> nn.Module:
    if model_type == dcgan:
        return DCGANGenerator(latent_dim, image_size)
    elif model_type == sagan:
        return SAGANGenerator(latent_dim, image_size)
    raise ValueError(f지원하지 않는 model_type: {model_type})

def build_discriminator(model_type: str, image_size: int) -> nn.Module:
    if model_type == dcgan:
        return DCGANDiscriminator(image_size)
    elif model_type == sagan:
        return SAGANDiscriminator(image_size)
    raise ValueError(f지원하지 않는 model_type: {model_type})

### Step 6 — Tool 함수

**`app/tools/gan_train_tool.py`**
```
- def train_gan(cmd: TrainGanCommand, on_sample: Callable[[int, bytes], None]) -> tuple[str, str, float, float]:
    # 반환: (g_weights_path, d_weights_path, final_g_loss, final_d_loss)
    # on_sample: 체크포인트 에폭에서 호출되는 콜백 (epoch, sample_png_bytes)

    device = torch.device(cmd.device if torch.cuda.is_available() else cpu)

    # 1. 모델 빌드 및 가중치 초기화
    G = build_generator(cmd.model_type, cmd.latent_dim, cmd.image_size).to(device)
    D = build_discriminator(cmd.model_type, cmd.image_size).to(device)
    G.apply(_weights_init)   # ConvTranspose2d: Normal(0, 0.02), BatchNorm: Normal(1, 0.02)
    D.apply(_weights_init)

    # 2. 데이터로더
    transform = transforms.Compose([
        transforms.Resize(cmd.image_size),
        transforms.CenterCrop(cmd.image_size),
        transforms.ToTensor(),
        transforms.Normalize([0.5]*3, [0.5]*3),   # -1~1 정규화
    ])
    dataset = ImageFolder(root=cmd.dataset_path, transform=transform)
    loader  = DataLoader(dataset, batch_size=cmd.batch_size, shuffle=True,
                         num_workers=4, pin_memory=True)

    # 3. 옵티마이저 / 손실
    criterion = nn.BCELoss()
    opt_G = optim.Adam(G.parameters(), lr=cmd.lr_g, betas=(0.5, 0.999))
    opt_D = optim.Adam(D.parameters(), lr=cmd.lr_d, betas=(0.5, 0.999))

    # 4. 고정 노이즈 (체크포인트 시각화용)
    fixed_z = torch.randn(16, cmd.latent_dim, 1, 1, device=device)

    # 5. 학습 루프
    g_loss_val = d_loss_val = 0.0
    real_label, fake_label = 1.0, 0.0

    for epoch in range(1, cmd.epochs + 1):
        for real_imgs, _ in loader:
            real_imgs = real_imgs.to(device)
            B = real_imgs.size(0)

            # --- D 업데이트 ---
            D.zero_grad()
            label_real = torch.full((B,), real_label, device=device)
            label_fake = torch.full((B,), fake_label, device=device)
            d_real = D(real_imgs)
            loss_d_real = criterion(d_real, label_real)

            z = torch.randn(B, cmd.latent_dim, 1, 1, device=device)
            fake_imgs = G(z).detach()
            d_fake = D(fake_imgs)
            loss_d_fake = criterion(d_fake, label_fake)

            loss_D = loss_d_real + loss_d_fake
            loss_D.backward()
            opt_D.step()

            # --- G 업데이트 ---
            G.zero_grad()
            z = torch.randn(B, cmd.latent_dim, 1, 1, device=device)
            fake_imgs = G(z)
            d_fake_for_g = D(fake_imgs)
            loss_G = criterion(d_fake_for_g, label_real)   # G는 D를 속이려 함
            loss_G.backward()
            opt_G.step()

            g_loss_val = loss_G.item()
            d_loss_val = loss_D.item()

        # 체크포인트 샘플 저장
        if epoch % cmd.save_interval == 0:
            with torch.no_grad():
                samples = G(fixed_z).cpu()
            sample_bytes = _tensor_grid_to_png(samples)
            on_sample(epoch, sample_bytes)

    # 가중치 임시 저장 후 경로 반환 (ModelAdapter.save에서 정본 이동)
    tmp_g = f/tmp/gan_G_{cmd.model_type}.pt
    tmp_d = f/tmp/gan_D_{cmd.model_type}.pt
    torch.save(G.state_dict(), tmp_g)
    torch.save(D.state_dict(), tmp_d)
    return tmp_g, tmp_d, g_loss_val, d_loss_val

- def _weights_init(m):
    classname = m.__class__.__name__
    if Conv in classname:
        nn.init.normal_(m.weight.data, 0.0, 0.02)
    elif BatchNorm in classname:
        nn.init.normal_(m.weight.data, 1.0, 0.02)
        nn.init.constant_(m.bias.data, 0)

- def _tensor_grid_to_png(tensor: torch.Tensor) -> bytes:
    # torchvision.utils.make_grid → PIL → PNG bytes
    grid = make_grid(tensor, normalize=True, value_range=(-1, 1))
    arr = (grid.permute(1,2,0).numpy() * 255).astype(np.uint8)
    buf = io.BytesIO()
    Image.fromarray(arr).save(buf, format=PNG)
    return buf.getvalue()

**`app/tools/gan_generate_tool.py`**
```
- _generators: dict[str, nn.Module] = {}   # model_type → 로드된 Generator 싱글턴
- _gen_lock = threading.Lock()

- def generate_images(
      weights_path: str,
      model_type: str,
      latent_dim: int,
      image_size: int,
      count: int,
      seed: int,
      device: str,
  ) -> list[bytes]:
    # 1. Generator 싱글턴 로드 (model_type별 캐시)
    with _gen_lock:
        if model_type not in _generators:
            G = build_generator(model_type, latent_dim, image_size)
            state = torch.load(weights_path, map_location=device)
            G.load_state_dict(state)
            G.to(device).eval()
            _generators[model_type] = G
        G = _generators[model_type]

    # 2. 노이즈 생성 (seed 고정)
    torch.manual_seed(seed)
    z = torch.randn(count, latent_dim, 1, 1).to(device)

    # 3. 이미지 생성
    with torch.no_grad():
        fake_imgs = G(z).cpu()

    # 4. 각 이미지를 PNG bytes로 변환
    result = []
    for i in range(count):
        img_tensor = fake_imgs[i]   # (3, H, W), 값 -1~1
        img_tensor = (img_tensor * 0.5 + 0.5).clamp(0, 1)   # 0~1 복원
        arr = (img_tensor.permute(1,2,0).numpy() * 255).astype(np.uint8)
        buf = io.BytesIO()
        Image.fromarray(arr).save(buf, format=PNG)
        result.append(buf.getvalue())
    return result

### Step 7 — Interactor

**`app/use_cases/gan_interactor.py`**
```
- GanInteractor(GanUseCase) 구현
- __init__: model: GanModelPort, graph: GanGraphPort, storage: GanStoragePort

- def train(self, cmd: TrainGanCommand) -> TrainGanResponse:
    # GPU-bound → def (동기), 라우터에서 asyncio.to_thread로 호출
    sample_urls: list[str] = []

    def on_sample(epoch: int, sample_bytes: bytes) -> None:
        # 동기 콜백 — asyncio 루프 없이 실행됨
        # S3 업로드는 여기서 동기적으로 처리 (boto3 동기 API)
        key = fgan/samples/{cmd.model_type}/epoch_{epoch:04d}.png
        url = self.storage.save_sample_sync(key, sample_bytes)   # 동기 S3 업로드
        sample_urls.append(url)

    tmp_g, tmp_d, g_loss, d_loss = train_gan(cmd, on_sample)
    weights_g = self.model.save_generator(tmp_g)
    weights_d = self.model.save_discriminator(tmp_d)

    return TrainGanResponse(
        weights_path_g=weights_g,
        weights_path_d=weights_d,
        epochs=cmd.epochs,
        final_g_loss=g_loss,
        final_d_loss=d_loss,
        sample_urls=sample_urls,
    )

- async def generate(self, cmd: GenerateCommand) -> GenerateResponse:
    weights_path = self.model.load_generator_path(cmd.model_type)
    seed = cmd.seed if cmd.seed is not None else int(time.time())

    # GPU-bound 생성 → 스레드풀 위임
    image_bytes_list: list[bytes] = await asyncio.to_thread(
        generate_images,
        weights_path, cmd.model_type, cmd.latent_dim,
        cmd.image_size, cmd.count, seed, cmd.device,
    )

    # S3 업로드 (I/O-bound → async)
    upload_tasks = [
        self.storage.save_generated_image(
            fgan/generated/{cmd.model_type}/{seed}_{i:04d}.png,
            img_bytes,
        )
        for i, img_bytes in enumerate(image_bytes_list)
    ]
    image_urls = list(await asyncio.gather(*upload_tasks))

    node_id = await self.graph.save_generation(
        cmd.model_type, cmd.count, image_urls, seed
    )
    return GenerateResponse(
        node_id=node_id,
        model_type=cmd.model_type,
        count=cmd.count,
        image_urls=image_urls,
        seed_used=seed,
    )

# D 업데이트

    loss_D = loss_d_real + loss_d_fake
scaler.scale(loss_D).backward()

with autocast():
