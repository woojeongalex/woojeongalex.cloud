> 대상: `woojeongai/apps/star_craft/`

> 완료 기준: 하네스 게이트(`ruff check`, `ruff format`, `mypy`) 전부 통과

## 0. 구현 전 필독

1. `woojeongai/CLAUDE.md` 전문을 읽고 4계층·ISP·DIP·Director 패턴을 숙지한다.
2. **패턴 정본** — `star_craft/dependencies/vision_provider.py`, `star_craft/adapter/outbound/redis/redis_crawl_target_repository.py`, `star_craft/adapter/outbound/resource_adapters/yolo/yolo_model_adapter.py` 를 반드시 먼저 읽어 기존 스타일을 맞춘다.
3. 기존 파일을 **절대 수정하지 않는다**. 신규 파일만 추가한다.
4. 모든 파일 상단에 `from __future__ import annotations` 를 붙인다.

---

## 1. 구현할 기능

ConvNeXt Nano(timm 라이브러리) 모델로 이미지를 분류하고,

- 분류 결과를 **Neo4j Graph DB** 에 노드로 저장한다.
- 후처리 태스크를 **Redis Queue** 에 적재한다.
- FastAPI `/api/star-craft/convnext/classify` 엔드포인트로 이미지를 수신한다.

## 2. 생성할 파일 목록 (순서 엄수)

아래 순서대로 파일을 **하나씩** 작성한다. 각 단계 완료 후 다음 단계로 넘어간다.

### Step 1 — Domain

**`domain/value_objects/classification_vo.py`**
```
- frozen dataclass
- 필드: label: str, confidence: float, top5: list[tuple[str, float]]
- 비즈니스 규칙: confidence는 0.0 ~ 1.0 사이여야 함 (__post_init__에서 검증)
```

### Step 2 — DTOs

**`app/dtos/convnext_dto.py`**
```
- ClassifyCommand(frozen dataclass): filename: str, content_type: str, image_bytes: bytes
- ClassifyResponse(frozen dataclass): node_id: str, label: str, confidence: float, queue_task_id: str
```

### Step 3 — Outbound Ports (ISP: 포트당 메서드 최소화)

**`app/ports/output/graph_classification_port.py`**
```python
class GraphClassificationPort(ABC):
    async def save_classification(
        self, filename: str, label: str, confidence: float
    ) -> str: ...  # 반환: node_id (str)
```

**`app/ports/output/classification_queue_port.py`**
```python
class ClassificationQueuePort(ABC):
    async def enqueue_post_process(
        self, node_id: str, label: str
    ) -> str: ...  # 반환: task_id (str)
```

### Step 4 — Inbound Port

**`app/ports/input/convnext_agent_use_case.py`**
```python
class ConvNextAgentUseCase(ABC):
    async def classify_and_store(self, cmd: ClassifyCommand) -> ClassifyResponse: ...
```

### Step 5 — Tool 함수 (CPU-bound 추론)

**`app/tools/__init__.py`** — 빈 파일

**`app/tools/convnext_classify_tool.py`**
```
- 함수명: classify_image(image_bytes: bytes) -> ClassificationVO
- 반환: domain/value_objects/classification_vo.py 의 ClassificationVO
- 모델: timm.create_model(convnext_nano, pretrained=True, num_classes=1000)
- 전처리: transforms.Resize(256) → CenterCrop(224) → ToTensor → Normalize([0.485,0.456,0.406],[0.229,0.224,0.225])
- 추론: torch.no_grad() 컨텍스트, softmax 후 top5 추출
- 모델은 모듈 레벨에서 1회 로드(싱글턴), thread-safe 보장 위해 _model_lock: threading.Lock 사용
- CPU-bound이므로 async 금지 — 반드시 def classify_image(...)
- 라벨: ImageNet 1000 클래스 (timm의 get_pretrained_cfg 또는 별도 labels.json 참조)
```

### Step 6 — Interactor

**`app/use_cases/convnext_agent_interactor.py`**
```
- ConvNextAgentInteractor(ConvNextAgentUseCase) 구현
- __init__: graph: GraphClassificationPort, queue: ClassificationQueuePort
- classify_and_store:
    1. asyncio.to_thread(classify_image, cmd.image_bytes)  ← 스레드풀 위임 필수
    2. await self.graph.save_classification(...)
    3. await self.queue.enqueue_post_process(...)
    4. ClassifyResponse 반환
- adapter/inbound 스키마 import 금지 (ISP)
```

### Step 7 — Outbound Adapters

**`adapter/outbound/neo4j/neo4j_classification_repository.py`**
```
- GraphClassificationPort 구현
- __init__: driver (neo4j.AsyncDriver)
- save_classification: Cypher MERGE 사용
  MERGE (n:ImageClassification {filename: $filename})
  SET n.label = $label, n.confidence = $confidence, n.updated_at = timestamp()
  RETURN elementId(n) AS node_id
- 반환: node_id (str)
- 로깅: logger = logging.getLogger(__name__), INFO 1줄
```

**`adapter/outbound/redis/redis_classification_queue.py`**
```
- ClassificationQueuePort 구현
- __init__: client (RedisClient) — core/matrix/redis_client.py 의 RedisClient 재사용
- REDIS_KEY = star_craft:convnext:post_process
- enqueue_post_process:
    payload = json.dumps({node_id: node_id, label: label, enqueued_at: ...})
    await asyncio.to_thread(self._client.rpush, REDIS_KEY, payload)
    task_id = ftask:{node_id}:{timestamp}
    return task_id
- 로깅: INFO 1줄
```

### Step 8 — Inbound Adapter

**`adapter/inbound/api/schemas/convnext_schema.py`**
```python
class ConvNextClassifyResponse(BaseModel):
    node_id: str
    label: str
    confidence: float
    queue_task_id: str
```

**`adapter/inbound/api/mappers/convnext_mapper.py`**
```
- schema_to_command(filename, content_type, data: bytes) -> ClassifyCommand
- response_to_schema(dto: ClassifyResponse) -> ConvNextClassifyResponse
```

**`adapter/inbound/api/v1/convnext_router.py`**
```
- prefix=/convnext, tags=[ConvNext]
- POST /classify:
    파라미터: file: UploadFile = File(...), use_case: ConvNextAgentUseCase = Depends(get_convnext_use_case)
    1. data = await file.read()
    2. cmd = schema_to_command(file.filename, file.content_type, data)
    3. result = await use_case.classify_and_store(cmd)
    4. return response_to_schema(result)
- 라우터에서 Repository·Driver 직접 생성 금지
```

**`adapter/inbound/api/deps/convnext_deps.py`**
```python
from star_craft.dependencies.convnext_provider import get_convnext_use_case
```

### Step 9 — Director (DIP 조립소)

**`dependencies/convnext_provider.py`**
```
- get_graph_repository() -> GraphClassificationPort:
    driver = neo4j.AsyncGraphDatabase.driver(NEO4J_URI, auth=(NEO4J_USER, NEO4J_PASSWORD))
    return Neo4jClassificationRepository(driver=driver)

- get_queue_repository() -> ClassificationQueuePort:
    return RedisClassificationQueue(client=RedisClient())

- get_convnext_use_case(
      graph: GraphClassificationPort = Depends(get_graph_repository),
      queue: ClassificationQueuePort = Depends(get_queue_repository),
  ) -> ConvNextAgentUseCase:
    return ConvNextAgentInteractor(graph=graph, queue=queue)

- 환경변수: NEO4J_URI, NEO4J_USER, NEO4J_PASSWORD (os.getenv 사용)
- get_graph_repository와 get_queue_repository를 반드시 분리 (CLAUDE.md §4 DIP 규칙)
```

## 4. 안티패턴 — 하지 말 것

```python
# ❌ 툴 함수에 async 붙이기 (CPU-bound 추론)
async def classify_image(image_bytes: bytes): ...

# ❌ Interactor에서 asyncio.to_thread 없이 classify_image 직접 호출
result = classify_image(cmd.image_bytes)  # 이벤트 루프 블로킹

# ❌ Director에서 get_graph_repository + get_queue_repository 미분리
def get_convnext_use_case(db=Depends(get_db)):
    return ConvNextAgentInteractor(
        graph=Neo4jClassificationRepository(...),
        queue=RedisClassificationQueue(...),
    )

# ❌ 모델을 요청마다 새로 로드
async def classify_and_store(self, cmd):
    model = create_model(convnext_nano, pretrained=True)  # 매 요청마다 로드 금지

# ❌ GraphPort와 QueuePort를 하나로 합치기 (ISP 위반)
class ClassificationPort(ABC):
    async def save_classification(...): ...
    async def enqueue_post_process(...): ...
```
