## 0. 구현 전 필독

1. `woojeongai/CLAUDE.md` 전문을 읽고 4계층·ISP·DIP·Director 패턴을 숙지한다.
2. **5번 지시서** `5.gan-image-generation-agent.md` 를 먼저 읽는다.
   GAN 모델 구조(`gan_models.py`)와 학습 패턴을 재사용한다.
3. 기존 파일을 **절대 수정하지 않는다**. 신규 파일만 추가한다.
4. 모든 파일 상단에 `from __future__ import annotations` 를 붙인다.
5. 이상 탐지 추론은 GPU-bound → `def` (동기), 호출 측에서 `asyncio.to_thread` 위임.

## 1. 이상 탐지 원리 (구현 전 숙지)

```
[학습 단계] — 정상 이미지만으로 GAN 학습
  정상 이미지 → Discriminator(D) + Generator(G) 학습

[추론 단계] — 입력 이미지가 정상인지 이상인지 판별
  입력 이미지 x
      ↓
  AnoGAN:  z* = argmin_z [Loss_R(x, G(z*)) + Loss_D(x, G(z*))]
           반복 최적화(back-prop through G)로 z* 탐색 → 느림
  EfficientGAN: Encoder(E)로 z = E(x) 직접 추정 → 빠름
      ↓
  이상 점수 = ||x - G(z*)|| + λ·||f(x) - f(G(z*))||
  (f: D의 중간 feature)
      ↓
  점수 > threshold → 이상(Anomaly), 이하 → 정상(Normal)
```

**AnoGAN vs EfficientGAN 선택 기준**

| | AnoGAN | EfficientGAN |
|---|---|---|
| 추론 속도 | 느림 (z 최적화 반복) | 빠름 (Encoder 1회 전파) |
| 정확도 | 높음 | AnoGAN에 근접 |
| 추가 학습 | G·D만 학습 | G·D·E 동시 학습 |
| 기본값 | — | ✅ 기본값 |

## 2. 구현할 기능

정상 이미지로 학습된 GAN으로 입력 이미지의 이상 여부를 판별하고,

- 이상 점수와 재구성 이미지를 **S3** 에 업로드한다.
- 탐지 결과(정상/이상, 점수)를 **Neo4j Graph DB** 에 노드로 저장한다.
- `POST /api/star-craft/anomaly/train` — 정상 이미지로 G·D·E 학습
- `POST /api/star-craft/anomaly/detect` — 이상 탐지 추론

### Step 5 — 모델 정의

**`app/tools/anomaly_models.py`**
```
- 5번의 gan_models.py 에서 DCGANGenerator, DCGANDiscriminator 를 재사용한다.
  from star_craft.app.tools.gan_models import DCGANGenerator, DCGANDiscriminator

- Encoder (EfficientGAN 전용):
class AnomalyEncoder(nn.Module):
    def __init__(self, latent_dim: int, image_size: int, channels: int = 3):
        # Discriminator와 동일한 다운샘플링 구조
        # 마지막 레이어: Conv2d → (latent_dim, 1, 1) → Flatten → Linear
        nef = 64
        self.conv = nn.Sequential(
            nn.Conv2d(channels, nef, 4, 2, 1, bias=False), nn.LeakyReLU(0.2, True),
            nn.Conv2d(nef, nef*2, 4, 2, 1, bias=False), nn.BatchNorm2d(nef*2), nn.LeakyReLU(0.2, True),
            nn.Conv2d(nef*2, nef*4, 4, 2, 1, bias=False), nn.BatchNorm2d(nef*4), nn.LeakyReLU(0.2, True),
            nn.Conv2d(nef*4, nef*8, 4, 2, 1, bias=False), nn.BatchNorm2d(nef*8), nn.LeakyReLU(0.2, True),
        )
        self.fc = nn.Linear(nef*8 * 4 * 4, latent_dim)

    def forward(self, x):
        h = self.conv(x).view(x.size(0), -1)
        return self.fc(h).unsqueeze(-1).unsqueeze(-1)   # (B, latent_dim, 1, 1)
```

**`app/tools/anomaly_detect_tool.py`**
```
- _loaded: dict = {}   # G, D, E 키로 싱글턴 캐시
- _detect_lock = threading.Lock()

- def _load_models(model_port, cmd):
    with _detect_lock:
        if G not in _loaded:
            G = DCGANGenerator(cmd.latent_dim, cmd.image_size)
            G.load_state_dict(torch.load(model_port.load_generator_path(), map_location=cmd.device))
            G.to(cmd.device).eval(); _loaded[G] = G
            D = DCGANDiscriminator(cmd.image_size)
            D.load_state_dict(torch.load(model_port.load_discriminator_path(), map_location=cmd.device))
            D.to(cmd.device).eval(); _loaded[D] = D
            e_path = model_port.load_encoder_path()
            if e_path:
                E = AnomalyEncoder(cmd.latent_dim, cmd.image_size)
                E.load_state_dict(torch.load(e_path, map_location=cmd.device))
                E.to(cmd.device).eval(); _loaded[E] = E
    return _loaded.get(G), _loaded.get(D), _loaded.get(E)

- def detect_anomaly(
      image_bytes: bytes,
      model_port: AnomalyModelPort,
      cmd: DetectAnomalyCommand,
  ) -> tuple[AnomalyVO, bytes]:
    # 반환: (AnomalyVO, reconstruction_png_bytes)

    G, D, E = _load_models(model_port, cmd)
    threshold = model_port.load_threshold()

    image = Image.open(io.BytesIO(image_bytes)).convert(RGB)
    transform = transforms.Compose([
        transforms.Resize(cmd.image_size), transforms.CenterCrop(cmd.image_size),
        transforms.ToTensor(), transforms.Normalize([0.5]*3, [0.5]*3),
    ])
    x = transform(image).unsqueeze(0).to(cmd.device)

    if E is not None:   # EfficientGAN: Encoder로 z 직접 추정
        with torch.no_grad():
            z_hat = E(x)
            x_hat = G(z_hat)
    else:               # AnoGAN: z 반복 최적화 (500 스텝)
        z_hat = _optimize_z(G, D, x, cmd.latent_dim, cmd.device, steps=500)
        with torch.no_grad():
            x_hat = G(z_hat)

    # 이상 점수 계산
    with torch.no_grad():
        recon_loss   = float(torch.mean(torch.abs(x - x_hat)).item())
        feat_x       = D.net[:-2](x)     # D의 마지막 Conv 이전 feature
        feat_xhat    = D.net[:-2](x_hat)
        feature_loss = float(torch.mean(torch.abs(feat_x - feat_xhat)).item())
    anomaly_score = recon_loss + 0.1 * feature_loss

    result = AnomalyResult(
        is_anomaly=anomaly_score > threshold,
        anomaly_score=anomaly_score,
        reconstruction_loss=recon_loss,
        feature_loss=feature_loss,
        threshold=threshold,
    )
    vo = AnomalyVO(filename=cmd.filename, result=result,
                   image_width=image.width, image_height=image.height)

    # 재구성 이미지 PNG bytes
    recon_bytes = _tensor_to_png(x_hat.squeeze(0).cpu())
    return vo, recon_bytes

- def _optimize_z(G, D, x, latent_dim, device, steps=500) -> torch.Tensor:
    # AnoGAN 전용: z 반복 최적화
    z = torch.randn(1, latent_dim, 1, 1, device=device, requires_grad=True)
    opt = optim.Adam([z], lr=0.01)
    for _ in range(steps):
        opt.zero_grad()
        x_hat = G(z)
        loss_r = torch.mean(torch.abs(x - x_hat))
        feat_x    = D.net[:-2](x)
        feat_xhat = D.net[:-2](x_hat)
        loss_d = torch.mean(torch.abs(feat_x - feat_xhat))
        (loss_r + 0.1 * loss_d).backward()
        opt.step()
    return z.detach()

- GPU-bound → def detect_anomaly(...) — async 절대 금지
```

### Step 7 — Interactor

**`app/use_cases/anomaly_interactor.py`**
```
- AnomalyInteractor(AnomalyUseCase) 구현
- __init__: model: AnomalyModelPort, graph: AnomalyGraphPort, storage: AnomalyStoragePort

- def train(self, cmd: TrainAnomalyCommand) -> TrainAnomalyResponse:
    tmp_g, tmp_d, tmp_e, threshold = train_anomaly_gan(cmd)
    g_path = self.model.save_generator(tmp_g)
    d_path = self.model.save_discriminator(tmp_d)
    e_path = self.model.save_encoder(tmp_e) if tmp_e else None
    self.model.save_threshold(threshold)
    return TrainAnomalyResponse(
        weights_path_g=g_path, weights_path_d=d_path,
        weights_path_e=e_path, threshold=threshold, epochs=cmd.epochs,
    )

- async def detect(self, cmd: DetectAnomalyCommand) -> DetectAnomalyResponse:
    (vo, recon_bytes) = await asyncio.to_thread(
        detect_anomaly, cmd.image_bytes, self.model, cmd
    )
    url = await self.storage.save_reconstruction(cmd.filename, recon_bytes)
    node_id = await self.graph.save_detection(
        cmd.filename, vo.result.is_anomaly, vo.result.anomaly_score,
        vo.result.threshold, url,
    )
    return DetectAnomalyResponse(
        node_id=node_id, filename=cmd.filename,
        is_anomaly=vo.result.is_anomaly, anomaly_score=vo.result.anomaly_score,
        threshold=vo.result.threshold, reconstruction_url=url,
    )
```
