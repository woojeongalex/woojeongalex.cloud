# 동영상 분류 에이전트 구현 지시서 (3DCNN → VideoMAE / TimeSformer)

> 대상: `woojeongai/apps/star_craft/`
> 패턴 정본: `woojeongai/CLAUDE.md` §3–5, `star_craft/dependencies/yolo_provider.py`, `star_craft/adapter/outbound/resource_adapters/yolo/yolo_model_adapter.py`
> 하드웨어: RTX 3050 8GB VRAM / RAM 16GB / i7-8700
> 완료 기준: 하네스 게이트(`ruff check`, `ruff format`, `mypy`) 전부 통과

## 0. 구현 전 필독

1. `woojeongai/CLAUDE.md` 전문을 읽고 4계층·ISP·DIP·Director 패턴을 숙지한다.
2. **패턴 정본** — 아래 파일을 반드시 먼저 읽어 기존 스타일을 맞춘다.
   - `star_craft/dependencies/yolo_provider.py`
   - `star_craft/adapter/outbound/resource_adapters/yolo/yolo_model_adapter.py`
   - `star_craft/app/use_cases/yolo_interactor.py`
   - `1.image-classfier-agent.md` — 이미지 분류 에이전트 패턴 재사용
3. 기존 파일을 **절대 수정하지 않는다**. 신규 파일만 추가한다.
4. 모든 파일 상단에 `from __future__ import annotations` 를 붙인다.
5. 동영상 분류 추론은 GPU-bound → `def` (동기), 호출 측에서 `asyncio.to_thread` 위임.

## 1. 모델 선택 — 3DCNN 대신 VideoMAE / TimeSformer

원서의 3DCNN(C3D)과 ECO는 구현 복잡도가 높고 RTX 3050 8GB에서 파인튜닝 한계.
Transformer 기반 최신 모델로 대체한다.

| 모델 | VRAM 추론 | VRAM 파인튜닝(QLoRA) | K400 Top-1 | HuggingFace | 기본값 |
|------|-----------|---------------------|-----------|-------------|--------|
| **VideoMAE-Base** | ~4GB | ~5GB (배치 4) | 81.5% | ✅ | ✅ |
| TimeSformer-Base | ~5GB | ~6GB (배치 2) | 77.9% | ✅ | — |
| Video Swin-T | ~6GB | ~6.5GB (배치 2) | 78.8% | ✅ | — |
| VideoMAE-Large | ~7GB | ⚠️ 배치 1만 가능 | 85.1% | ✅ | — |
| 3D ResNet-50 (원서) | ~3GB | ~4GB (배치 8) | 74.9% | ✅ | — |

> **RTX 3050 8GB 추천**: VideoMAE-Base — 정확도·VRAM 균형이 가장 좋음.
> **최고 정확도 원하면**: VideoMAE-Large (배치 1, 다른 프로세스 종료 필요).

## 2. 동영상 전처리 핵심 (구현 전 숙지)

```
동영상 파일 (.mp4, .avi 등)
    ↓
프레임 추출: 균일 샘플링 (N프레임)
    ↓
각 프레임 리사이즈: 224×224
    ↓
정규화: mean=[0.485,0.456,0.406], std=[0.229,0.224,0.225]
    ↓
텐서: (1, N, C, H, W) = (batch, frames, channels, height, width)
    ↓
VideoMAE 입력 (N=16 프레임이 기본)
```

## 3. 구현할 기능

VideoMAE-Base로 동영상 클립의 행동/장면을 분류하고,

- 분류 결과(레이블 + 신뢰도 top-5)를 **Neo4j Graph DB** 에 노드로 저장한다.
- 분류된 동영상을 **S3** 에 업로드한다.
- `POST /api/star-craft/video/classify` — 동영상 파일 업로드 후 분류
- `POST /api/star-craft/video/train` — 커스텀 데이터셋으로 파인튜닝

**`app/tools/video_frame_extractor.py`**
```
- def extract_frames(video_bytes: bytes, num_frames: int) -> tuple[list[np.ndarray], float]:
    # 반환: (프레임 리스트, 동영상 길이_초)

    import tempfile, cv2

    # bytes → 임시 파일 (OpenCV는 파일 경로 필요)
    with tempfile.NamedTemporaryFile(suffix=.mp4, delete=False) as tmp:
        tmp.write(video_bytes)
        tmp_path = tmp.name

    cap = cv2.VideoCapture(tmp_path)
    total_frames = int(cap.get(cv2.CAP_PROP_FRAME_COUNT))
    fps          = cap.get(cv2.CAP_PROP_FPS) or 30.0
    duration     = total_frames / fps

    # 균일 샘플링
    indices = np.linspace(0, total_frames - 1, num_frames, dtype=int)
    frames  = []
    for idx in indices:
        cap.set(cv2.CAP_PROP_POS_FRAMES, int(idx))
        ret, frame = cap.read()
        if ret:
            frame_rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
            frames.append(frame_rgb)
    cap.release()
    os.unlink(tmp_path)

    return frames, duration

- def (동기) — I/O + CPU 혼합이지만 동기 처리
```

**`app/tools/videomae_classify_tool.py`**
```
- 싱글턴 패턴:
    _processor = None
    _model = None
    _model_lock = threading.Lock()

- def _get_model_and_processor(model_name: str):
    global _processor, _model
    with _model_lock:
        if _model is None:
            from transformers import VideoMAEImageProcessor, VideoMAEForVideoClassification
            _processor = VideoMAEImageProcessor.from_pretrained(model_name)
            _model = VideoMAEForVideoClassification.from_pretrained(model_name)
            device = cuda if torch.cuda.is_available() else cpu
            _model.to(device).eval()
    return _processor, _model

- def classify_video(
      video_bytes: bytes,
      model_name: str,
      num_frames: int,
      device: str,
  ) -> VideoClassificationVO:

    # 1. 프레임 추출
    frames, duration = extract_frames(video_bytes, num_frames)

    # 2. 전처리
    processor, model = _get_model_and_processor(model_name)
    inputs = processor(list(frames), return_tensors=pt).to(model.device)
    # inputs[pixel_values]: (1, num_frames, 3, 224, 224)

    # 3. 추론
    with torch.no_grad():
        outputs = model(**inputs)

    # 4. top-5 추출
    probs = torch.softmax(outputs.logits[0], dim=0)
    top5_vals, top5_idxs = torch.topk(probs, 5)
    top5 = [
        VideoLabel(
            label=model.config.id2label[int(idx)],
            confidence=float(val),
        )
        for idx, val in zip(top5_idxs, top5_vals)
    ]

    return VideoClassificationVO(
        top1_label=top5[0].label,
        top1_confidence=top5[0].confidence,
        top5=top5,
        frame_count=len(frames),
        duration_seconds=duration,
        model_name=model_name,
    )

- GPU-bound → def classify_video(...) — async 절대 금지
```

**`app/tools/video_train_tool.py`**
```
- def train_video_classifier(cmd: TrainCommand) -> tuple[str, float, list[str]]:
    # 반환: (tmp_dir, eval_accuracy, label_names)

    from transformers import (
        VideoMAEForVideoClassification, VideoMAEImageProcessor,
        TrainingArguments, Trainer,
    )
    from torch.utils.data import Dataset as TorchDataset
    import evaluate

    # 커스텀 데이터셋 구조:
    # dataset_path/
    #   train/<class_name>/*.mp4
    #   val/<class_name>/*.mp4

    processor   = VideoMAEImageProcessor.from_pretrained(cmd.model_name)
    label_names = sorted(os.listdir(os.path.join(cmd.dataset_path, train)))
    label2id    = {name: i for i, name in enumerate(label_names)}

    class VideoDataset(TorchDataset):
        def __init__(self, root: str):
            self.samples = []
            for label_name in label_names:
                folder = os.path.join(root, label_name)
                for f in os.listdir(folder):
                    if f.endswith((.mp4, .avi)):
                        self.samples.append((os.path.join(folder, f), label2id[label_name]))

        def __len__(self): return len(self.samples)

        def __getitem__(self, idx):
            path, label = self.samples[idx]
            with open(path, rb) as f:
                video_bytes = f.read()
            frames, _ = extract_frames(video_bytes, cmd.num_frames)
            inputs = processor(frames, return_tensors=pt)
            return {
                pixel_values: inputs[pixel_values].squeeze(0),
                labels: torch.tensor(label),
            }

    model = VideoMAEForVideoClassification.from_pretrained(
        cmd.model_name,
        num_labels=cmd.num_labels,
        label2id=label2id,
        id2label={i: n for n, i in label2id.items()},
        ignore_mismatched_sizes=True,
    )

    training_args = TrainingArguments(
        output_dir=/tmp/videomae_finetuned,
        per_device_train_batch_size=cmd.batch_size,   # 4 (RTX 3050 8GB)
        gradient_accumulation_steps=4,                # 유효 배치 16
        num_train_epochs=cmd.epochs,
        learning_rate=5e-5,
        fp16=True,
        optim=paged_adamw_8bit,
        dataloader_num_workers=4,
        evaluation_strategy=epoch,
        save_strategy=epoch,
        load_best_model_at_end=True,
        metric_for_best_model=accuracy,
        remove_unused_columns=False,
    )

    acc_metric = evaluate.load(accuracy)

    def compute_metrics(eval_pred):
        logits, labels = eval_pred
        preds = logits.argmax(axis=-1)
        return acc_metric.compute(predictions=preds, references=labels)

    train_ds = VideoDataset(os.path.join(cmd.dataset_path, train))
    val_ds   = VideoDataset(os.path.join(cmd.dataset_path, val))

    trainer = Trainer(
        model=model, args=training_args,
        train_dataset=train_ds, eval_dataset=val_ds,
        compute_metrics=compute_metrics,
    )
    trainer.train()
    metrics = trainer.evaluate()

    tmp_dir = /tmp/videomae_best
    trainer.save_model(tmp_dir)
    processor.save_pretrained(tmp_dir)

    return tmp_dir, metrics.get(eval_accuracy, 0.0), label_names
```

```
- VideoInteractor(VideoUseCase) 구현
- __init__: model: VideoModelPort, graph: VideoGraphPort, storage: VideoStoragePort

- async def classify(self, cmd: ClassifyVideoCommand) -> ClassifyVideoResponse:
    1. vo = await asyncio.to_thread(
           classify_video,
           cmd.video_bytes, cmd.model_name, cmd.num_frames, cmd.device,
       )
    2. video_url = await self.storage.save_video(cmd.filename, cmd.video_bytes)

           top5_dict, video_url, vo.duration_seconds,
       )

           node_id=node_id, filename=cmd.filename,

    return TrainResponse(
        weights_path=weights_path, epochs=cmd.epochs,
```

```python
from transformers import BitsAndBytesConfig
from peft import LoraConfig, get_peft_model, prepare_model_for_kbit_training

bnb_config = BitsAndBytesConfig(
    load_in_4bit=True,
    bnb_4bit_quant_type=nf4,
    bnb_4bit_compute_dtype=torch.float16,
    bnb_4bit_use_double_quant=True,

model = VideoMAEForVideoClassification.from_pretrained(
    cmd.model_name,
    quantization_config=bnb_config,
    num_labels=cmd.num_labels,
```
