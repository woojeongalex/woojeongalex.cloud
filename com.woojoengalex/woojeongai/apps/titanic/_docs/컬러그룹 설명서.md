# 🎨 그래프 뷰 컬러 그룹 설명서

Obsidian 전역 그래프 뷰에서 노드를 색상으로 구분한 규칙을 정리한 문서입니다.  
설정 파일 위치: `.obsidian/graph.json` → `colorGroups`

---

## 색상 구성표

| 색상 | 그룹 | 해당 경로/파일 | 설명 |
|------|------|----------------|------|
| 🟡 노랑 | 수업 내용 | `vault/수업 내용/` | 강의 일자별 정리 노트 |
| 🔵 파랑 | DevOps 문서 | `vault/DevOPs/` | 아키텍처·백엔드·프론트 규칙 문서 |
| 🔴 빨강 | Titanic | `path:titanic` / `file:TITANIC` / `file:타이타닉` | 타이타닉 레퍼런스 앱 관련 전체 |
| 🟣 보라 | Music | `path:music` | 음악 앱 관련 파일 |
| 🟠 주황 | Friday13th | `path:friday13th` | 인증 앱 관련 파일 |
| 🟢 초록 | 기타 Vault | `path:vault` | 위 규칙에 해당하지 않는 나머지 노트 |
| ⬜ 회색 | CLAUDE | `file:CLAUDE` | Claude 행동 하네스·인수인계 문서 |

---

## 우선순위 규칙

> 위쪽 그룹이 아래쪽보다 **우선 적용**됩니다.

```
수업 내용 (🟡) > DevOPs (🔵) > Titanic (🔴) > Music (🟣) > Friday13th (🟠) > vault 전체 (🟢) > CLAUDE (⬜)
```

예시: `vault/DevOPs/Backend/TITANIC_ERD.md`는 titanic 관련이지만,  
`path:vault/DevOPs` 규칙이 더 위에 있으므로 **파랑(🔵)** 으로 표시됩니다.

---

## 설정 변경 방법

`.obsidian/graph.json`의 `colorGroups` 배열을 수정합니다.  
색상 값은 **RGB 정수** (예: `16750848` = `#FFB800` 노랑).

```json
{
  "query": "path:vault/수업 내용",
  "color": { "a": 1, "rgb": 16750848 }
}
```

변경 후 Obsidian에서 그래프 뷰를 **닫고 다시 열면** 반영됩니다.
